# 子目录站点


网站若存放在子目录，如 `http://yoursite.com/blog`

## 站点配置 （非子目录站点请填 **`/`**）

<blockquote class="note">
    相关链接: <a href="https://hexo.io/docs/configuration.html#URL" target="_blank">https://hexo.io/docs/configuration.html#URL</a>
</blockquote>

```
root: /blog/
```


## 主题配置 （非子目录站点请 **`留空`**）

```
root_url: /blog/
```
