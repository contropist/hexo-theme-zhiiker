# 语言切换

https://hexo.io/docs/internationalization.html

<blockquote class='note'> 在站点配置中填写对应语言的代码，留空默认显示简体中文，但建议指定语言以免显示有误</blockquote>

```yaml
# Hexo Configuration
## Docs: http://hexo.io/docs/configuration.html

# Site
language: zh-Hans
```

![语言切换](/src/i18n-setup.png)
