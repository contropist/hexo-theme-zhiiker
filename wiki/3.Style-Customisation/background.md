# 网页背景图

![背景](/src/feat.background.jpg)

背景图文件所在路径: 

```
/yelee/source/background/
```

`themes/yelee/_config.yml` 背景参数:

```yaml
background_image: 5
```

- 默认值为5，可按需修改

- "5": 设置`/yelee/source/background/`文件夹中 `bg-1.jpg` 到 `bg-5.jpg` 这5张图片为背景

- "0": 取消网页背景图，使用淳朴的灰白主题
