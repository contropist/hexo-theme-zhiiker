# 代码高亮风格:

通过 `code_block` 切换内置代码高亮配色主题，目前额外提供 4 种搭配

```yaml
## 移除 `#` 后自定义样式生效
highlight_style:
  #on: true
  code_block: 1  # Value: 0 - 4 
```
