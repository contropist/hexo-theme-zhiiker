# 图片浏览器

默认使用 [fancyBox](http://fancyapps.com/fancybox/) 作为文章内的图片浏览器

``` yaml
fancybox: true
```

指定文章中关闭图片浏览器 `fancybox: false`
```yaml
title: Hello World
date: 2015-08-19 00:00:00
fancybox: false
---
```
<blockquote class="note">
    文章内 fancybox 设置的优先级高于主题默认设置
</blockquote>
