# GitHub 仓库展示小挂件

> GitHub Repo Widget: https://github.com/hustcc/GitHub-Repo-Widget.js

```
github_widget: true
```

开启后在正文中插入如下代码即可

```html
<div class="github-widget" data-repo="<GitHub 用户名>/<仓库名>.js"></div>

<!-- e.g. -->
<div class="github-widget" data-repo="MOxFIVE/hexo-theme-yelee"></div>
```
